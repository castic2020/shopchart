# 系统功能
一种智能购物车，其主要功能如下：    
  1、利用麦克纳姆轮可全方位行车，避免空间小、拐弯不便的问题。  
  2、利用图像识别、二维码追踪等使购物车可以跟着顾客走。  
  3、利用摄像头寻找道路自动前进。
